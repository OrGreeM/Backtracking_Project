# Пошук рiшень з використанням алгоритму Backtracking

# Backtracking — Розв'язання задач пошуку та комбінаторної оптимізації

## 📌 Про проєкт

Backtracking — потужний підхід до розв'язання задач пошуку та комбінаторної оптимізації, який базується на ідеї пошуку з поверненням та вчасному обрізанні гілок дерева рішень. Його широко застосовують для задач на перебір, побудову рішень на основі обмежень, розфарбовування графів, планування та багато іншого.

---

## 🎯 Цілі

- Навчитись реалізовувати алгоритм backtracking
- Застосувати його до практичних задач
- Ознайомитись з оптимізаціями: forward checking, MRV, constraint propagation
- Пов'язати ідеї backtracking з базовими поняттями дискретної математики: графами, комбінаторикою, множинами, булевими обмеженнями

---

## 📋 Етап 1. Базові задачі

Реалізувати backtracking-рішення для наступних задач:

### 1. Розв'язання Судоку 9×9

Розв'язання Судоку з різною кількістю початково заповнених клітинок.

**Функціональність:**
- Розв'язання Судоку 9×9 з різною складністю
- Генерування випадкових полів
- Порівняння ефективності з жадібними/DFS-варіантами

### 2. Вихід з лабіринту

Знаходження шляху з точки A до точки B у прямокутному лабіринті.

**Функціональність:**
- Знаходження шляху з точки A до точки B
- Візуалізація алгоритму
- Оптимізація шляху

### 3. Кросворд

Автоматичне заповнення заданої сітки.

**Функціональність:**
- Автоматичне заповнення сітки кросворду
- Використання списку слів
- Підтримка горизонтальних і вертикальних слів

---

## 🎨 Етап 2. Розширені задачі


### 1. Розфарбовування графа (k-розфарбування)

Розфарбуйте неорієнтований граф G = (V,E) у k кольорів так, щоб сусідні вершини мали різні кольори.

### 2. N-Queens

Розмістіть N ферзів на шахівниці N ×N так, щоб жодні два не били одне одного.
