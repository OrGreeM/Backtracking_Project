## 📌 Про проєкт

Алгоритм **Backtracking** (пошук із поверненням) є фундаментальним методом у дискретній математиці та комп'ютерних науках для повного або часткового перебору варіантів у дереві рішень. Основною перевагою нашої реалізації є застосування сучасних евристик та методів оптимізації, які дозволяють відтинати неефективні гілки рішень (pruning) на ранніх етапах пошуку, що значно підвищує швидкість у порівнянні з класичним повним перебором.

### 🌟 Ключові оптимізації, застосовані в проєкті:
*   **Forward Checking (FC / Випереджальний пошук):** Динамічне звуження простору пошуку шляхом оновлення доменів майбутніх змінних.
*   **Minimum Remaining Values (MRV):** Евристика вибору змінної з найменшою кількістю валідних варіантів для заповнення (стратегія "найбільш обмеженого першого").
*   **Constraint Propagation:** Поширення обмежень для виявлення тупикових станів до початку безпосереднього перебору.
*   **Heuristic Search (A*, Manhattan Distance):** Поєднання комбінаторного пошуку з геометричними та просторовими евристиками.

## ✨ Що було зроблено

Проєкт розділено на **5 високофункціональних модулів**, кожен з яких повністю вирішує окрему складну задачу:

### 1. 🧩 Кросворди (Crossword CSP)
Перетворення побудови кросворду на класичну задачу CSP, де кожна клітинка та слово пов'язані взаємними обмеженнями на перетин літер.
*   **Реалізовані алгоритми:** `Basic Backtracking`, `Forward Checking (FC)`, `MRV` та комбінований `MRV + FC`.
*   **Генерація:** Автоматичне створення випадкових сіток розміром $N \times N$ із можливістю задання частки заблокованих клітинок (block ratio) та автозаповненням на основі великого вбудованого англійського словника (`en.txt`).
*   **Візуалізація:** Крок за кроком відображає підбір слів у реальному часі за допомогою графічного рендерингу в `matplotlib`.

### 2. 🌀 Лабіринти (Maze Pathfinding)
Генерація та розв'язання складних лабіринтів.
*   **Генератор:** Реалізовано алгоритм **Randomized DFS** для автоматичної побудови ідеальних лабіринтів (без петель та ізольованих областей) будь-якого непарного розміру.
*   **Реалізовані пошукові алгоритми:**
    *   `dfs` — Класичний backtracking (глибина).
    *   `smart` — Оптимізований пошук у глибину, що пріоритезує рух у напрямку цілі за Мангеттенською відстанню.
    *   `bfs` — Пошук у ширину (пошук гарантованого найкоротшого шляху).
    *   `astar` — Алгоритм $A^*$ з евристичною оцінкою відстані до фінішу.
*   **Порівняльний аналіз:** Автоматична побудова бенчмарк-графіків часу виконання, кількості кроків та довжини фінального шляху.

### 3. 🔢 Судоку (Sudoku Solver)
Аналізатор та генератор класичних головоломок Судоку 9×9.
*   **Реалізовані алгоритми:** `Pure Backtracking` (наївний перебір), `MRV Heuristic`, `MRV + FC` (миттєве розв'язання) та `Greedy Solver` (жадібний пошук).
*   **Генератор:** Генерація унікальних повноцінних ігрових дощок із валідацією наявності єдиного розв'язку для складностей: *Easy, Medium, Hard, Expert*.
*   **Стрес-тести:** Підтримка завантаження надскладних екстремальних судоку (наприклад, відомого тесту `Escargot`).

### 4. ♛ Задача N Ферзів (N-Queens)
Класичне розташування $N$ взаємно безпечних ферзів на дошці $N \times N$.
*   **Візуалізація:** Унікальний графічний режим, який покроково анімує як заповнення дошки, так і паралельну розбудову дерева рішень (backtracking tree) за допомогою бібліотеки `matplotlib`.
*   **Бенчмаркінг:** Аналіз залежності часу пошуку від розміру дошки $N$ з побудовою графіків порівняння оптимізованих алгоритмів.

### 5. 🎨 Розфарбовування Графів (Graph Coloring)
Задача розподілу $K$ кольорів між вершинами неорієнтованого графа так, щоб жодні дві суміжні вершини не мали однакового кольору.
*   **Реалізовані алгоритми:** Простий backtracking, оптимізований backtracking (з інтегрованими евристиками MRV, Forward Checking та LCV — Least Constraining Value), а також базові референсні евристики.
*   **Генерація:** Генератор випадкових графів за моделлю Ердеша-Реньї ($N$ вершин, ймовірність ребра $p$).
*   **Імпорт:** Можливість зчитувати кастомні матриці суміжності з текстових файлів.
*   **Аналіз пам'яті:** Повноцінний профілювальник пам'яті через `tracemalloc` та генерація детальних аналітичних звітів у форматі Markdown та PNG.

## 🚀 Як запустити код

### 1. Попередні вимоги
Переконайтеся, що на вашому комп'ютері встановлено **Python 3.10** або новішої версії з інстальованим менеджером пакетів `pip`.

### 2. Встановлення залежностей
Встановіть графічні та обчислювальні бібліотеки за допомогою утиліти `pip` у вашому терміналі:

```bash
pip install matplotlib numpy
```

### 3. Запуск програми

Проєкт підтримує два способи взаємодії:

#### Варіант А. Інтерактивне консольне меню (Рекомендовано 🌟)
Просто запустіть головний хаб без додаткових аргументів. Перед вами з'явиться кольорове та зручне текстове меню, яке проведе вас через усі налаштування кожного модуля:

```bash
python cli.py
```

#### Варіант Б. Прямі CLI-команди з параметрами (Для розробників/бенчмаркінгу)
Ви можете миттєво запускати окремі модулі з кастомними параметрами безпосередньо з вашого терміналу:

> [!TIP]
> Кожна підкоманда підтримує детальний хелпер. Запустіть `python cli.py --help` або `python cli.py [subcommand] --help` для перегляду всіх доступних прапорців.

```bash
# Розв'язати надскладний вбудований судоку Escargot за допомогою алгоритму MRV+FC

# Згенерувати судоку рівня складності expert та вивести його рішення
python cli.py sudoku generate --difficulty expert --show-solution

# Запустити порівняльне тестування на базі легких та середніх судоку
```

```bash

# Порівняти ефективність алгоритмів для N=8 без показу анімації кроків
```

```bash
# Спробувати розфарбувати випадковий граф з 15 вершинами (p=0.3) у K=4 кольори

# Зчитати матрицю суміжності з файлу та знайти розфарбування у K=3 кольори
python cli.py coloring --file c:/path/to/matrix.txt --k 3
```
