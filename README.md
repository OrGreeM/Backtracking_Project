Програмний комплекс для моделювання, візуалізації та аналізу NP-повних комбінаторних задач і задач задоволення обмежень (CSP — Constraint Satisfaction Problems), розв'язаних за допомогою алгоритму пошуку з поверненням (Backtracking) та його оптимізованих варіантів.

Усі модулі об'єднані під єдиною консольною оболонкою `cli.py`, яка підтримує інтерактивне меню та гнучке передавання аргументів командного рядка.

## Про проєкт

Алгоритм Backtracking є фундаментальним методом у дискретній математиці та комп'ютерних науках для перебору варіантів у дереві рішень. Ключова перевага реалізації — застосування сучасних евристик і методів оптимізації, що дозволяють відтинати неперспективні гілки (pruning) на ранніх етапах пошуку, суттєво прискорюючи роботу порівняно з наївним повним перебором.

### Ключові оптимізації

- **Forward Checking (FC)** — динамічне звуження простору пошуку шляхом оновлення доменів майбутніх змінних після кожного присвоєння.
- **Minimum Remaining Values (MRV)** — евристика вибору змінної з найменшою кількістю допустимих значень ("найбільш обмежений першим").
- **Constraint Propagation** — поширення обмежень для виявлення тупикових станів до початку безпосереднього перебору.
- **Heuristic Search (A\*, Manhattan Distance)** — поєднання комбінаторного пошуку з геометричними евристиками.

## Модулі проєкту

### 1. Кросворди (Crossword CSP)

Побудова кросворду сформульована як задача CSP, де кожне слово-слот є змінною, а спільні клітинки на перетині рядків і стовпців утворюють обмеження на літери.

- **Алгоритми:** Basic Backtracking, Forward Checking (FC), MRV, MRV + FC.
- **Генерація:** автоматичне створення випадкових сіток N x N із налаштованою часткою заблокованих клітинок і заповненням зі вбудованого англійського словника (`en.txt`).
- **Візуалізація:** покроковий рендеринг підбору слів у реальному часі через `matplotlib` з кнопками керування і регулятором швидкості.

### 2. Лабіринти (Maze Pathfinding)

Генерація та розв'язання складних лабіринтів.

- **Генератор:** алгоритм Randomized DFS для побудови ідеальних лабіринтів (без петель та ізольованих областей) довільного непарного розміру.
- **Алгоритми пошуку:**
  - `dfs` — класичний backtracking (пошук у глибину).
  - `smart` — евристичний DFS, що пріоритезує рух у напрямку виходу за Манхеттенською відстанню.
  - `bfs` — пошук у ширину, що гарантує знаходження найкоротшого шляху.
  - `astar` — алгоритм A* з евристичною оцінкою до фінішу.
- **Бенчмаркінг:** автоматична побудова графіків часу виконання, кількості кроків і довжини знайденого шляху для всіх алгоритмів.

### 3. Судоку (Sudoku Solver)

Розв'язувач і генератор класичних головоломок Судоку 9×9.

- **Алгоритми:** Pure Backtracking (наївний перебір), MRV Heuristic, MRV + FC, Greedy Solver.
- **Генератор:** створення унікальних ігрових дощок із валідацією єдиності розв'язку для рівнів складності Easy, Medium, Hard, Expert.
- **Стрес-тести:** підтримка завантаження надскладних пресетів (зокрема відомого тесту Escargot).

### 4. Задача N Ферзів (N-Queens)

Класичне розміщення N взаємно безпечних ферзів на дошці N x N.

- **Візуалізація:** покрокова анімація розміщення ферзів з одночасним динамічним малюванням дерева рішень у реальному часі через `matplotlib`.
- **Бенчмаркінг:** аналіз залежності часу пошуку від розміру дошки N з побудовою порівняльних графіків для різних алгоритмів.

### 5. Розфарбовування графів (Graph Coloring)

Розподіл K кольорів між вершинами неорієнтованого графа так, щоб жодні дві суміжні вершини не мали однакового кольору.

- **Алгоритми:** простий backtracking; оптимізований backtracking з евристиками MRV, Forward Checking та LCV (Least Constraining Value); базові референсні евристики.
- **Генерація:** генератор випадкових графів за моделлю Ердеша-Реньї (N вершин, ймовірність ребра p).
- **Імпорт:** зчитування матриці суміжності з текстового файлу.
- **Аналіз пам'яті:** профілювання через `tracemalloc` та автоматична генерація звітів у форматі Markdown і PNG.

## Як запустити

### Вимоги

Python 3.10 або новіший з менеджером пакетів `pip`.

### Встановлення залежностей

```bash
pip install matplotlib numpy
```

### Запуск

#### Варіант А. Інтерактивне меню (рекомендовано)

```bash
python cli.py
```

#### Варіант Б. CLI з параметрами

> Кожна підкоманда підтримує довідку: `python cli.py --help` або `python cli.py <підкоманда> --help`.

<details>
<summary>Приклади CLI-команд для кожного модуля</summary>

##### Кросворди

```bash
# Випадковий кросворд 5x5 з 25% блоків
python cli.py crossword --grid-size 5 --block-ratio 0.25 --seed 42

# Порівняння лише MRV та MRV+FC без візуалізації
python cli.py crossword --no-vis --algs mrv mrv_fc
```

##### Лабіринти

```bash
# Розв'язати лабіринт 41x41 алгоритмом A*
python cli.py maze solve --size 41 --algorithm astar --show-maze

# Бенчмарк усіх алгоритмів на лабіринті 101x101
python cli.py maze benchmark --size 101
```

##### Судоку

```bash
# Розв'язати Escargot алгоритмом MRV+FC
python cli.py sudoku solve --builtin escargot --algo mrv_fc

# Згенерувати судоку рівня expert і вивести рішення
python cli.py sudoku generate --difficulty expert --show-solution

# Порівняльне тестування на легких та середніх головоломках
python cli.py sudoku benchmark --difficulties easy medium
```

##### N-Queens

```bash
# Покрокова візуалізація для N=6
python cli.py nqueens visualize --size 6

# Порівняння алгоритмів для N=8 без анімації
python cli.py nqueens compare --size 8 --no-vis
```

##### Розфарбовування графів

```bash
# Випадковий граф із 15 вершин, ймовірність ребра 0.3, K=4 кольори
python cli.py coloring --n 15 --p 0.3 --k 4

# Граф із файлу матриці суміжності, K=3 кольори
python cli.py coloring --file my_graph.txt --k 3
```
